## H-Tag Structure Rules

Every page must follow this hierarchy strictly:

```
H1 — One per page only. Must contain primary keyword + "Vadodara"
     Example: "AC Repair in Vadodara — Fast & Reliable Service"

H2 — Section headings. Include secondary keywords.
     Examples: "Why Choose Jyoti AC?", "Our AC Services in Vadodara",
               "Areas We Serve in Vadodara"

H3 — Subsections, FAQ questions, individual service items.
     Examples: "Split AC Repair", "Window AC Repair", "What is AC AMC?"

H4/H5 — Use sparingly for deeply nested content only.
```

**Rule:** Never skip a heading level. Never use H1 twice.

## Image SEO Rules

Every image on the site must have:

1. **File name** — keyword-based, lowercase, hyphenated
   - ✅ `ac-repair-vadodara-technician.jpg`
   - ❌ `IMG_20240312_1142.jpg`

2. **ALT text** — descriptive + keyword where relevant
   - ✅ `AC technician repairing split AC unit in Vadodara`
   - ❌ `image1` or empty

3. **Format** — Use WebP for all images (convert on upload)

4. **Size** — Hero images max 150KB, content images max 80KB

5. **Dimensions** — Define width/height attributes in HTML to prevent CLS

## On-Page SEO Checklist (Per Page)

```
[ ] H1 contains primary keyword + "Vadodara"
[ ] Meta title ≤ 60 characters
[ ] Meta description ≤ 160 characters, includes phone number
[ ] Primary keyword in first 100 words of body text
[ ] At least 3–5 area keywords mentioned naturally in body
[ ] Internal links to 3+ related pages with keyword anchors
[ ] All images have file names + ALT text
[ ] Schema markup added (Service + FAQ + Breadcrumb)
[ ] Page URL is lowercase, hyphenated, keyword-rich
[ ] Mobile responsive verified
[ ] Page speed < 3s on mobile (Google PageSpeed)
[ ] No keyword stuffing — reads naturally
[ ] Call to action above the fold
[ ] Phone number in text (crawlable, not just image)
```
